Implicit-Association Test (IAT)
  • IAT inspired program to identify implicit ethnicity bias through "trolley dilemma" decisions. [link to code]
    • Timed basic trolley dillema with the possibility to assign specific ethnicity. Option to play the bridge version (or "fat man"). Written with OpenSesame.




Stata package
  • plot_poly - Post estimation Stata function to plot polynomial dose-response [link to code]
    • Rely on Stata built-in predictnl function to compute marginal effect when the the dose-response follow a polynomial form (of any order). Option to add an histogram of the underlying support. User can set the reference point and decides to plot either in-sample or out of the data support. The later is relevant for any counterfactual analysis and projected climate impact in particular.
